# Exercise Rep8: Weyl Denominator Formula for sl₂

## Problem Statement

> [!question] Problem
> The **Weyl denominator formula** states that for a semisimple Lie algebra $\mathfrak{g}$:
> $$\sum_{w \in W} \epsilon(w) e^{w(\rho)} = e^{\rho} \prod_{\alpha \in \Delta^+} (1 - e^{-\alpha})$$
>
> where $W$ is the Weyl group, $\epsilon(w) = (-1)^{\ell(w)}$, $\rho = \frac{1}{2}\sum_{\alpha > 0} \alpha$, and $\Delta^+$ is the set of positive roots.
>
> **Part (a):** For $\mathfrak{sl}_2(\mathbb{C})$, identify the root system, Weyl group, and compute $\rho$.
>
> **Part (b):** Verify the Weyl denominator formula directly for $\mathfrak{sl}_2$.
>
> **Part (c):** Use the character formula to derive $\text{ch } V_n = z^n + z^{n-2} + \cdots + z^{-n}$ for the $(n+1)$-dimensional representation.

## Hints

> [!hint]- Hint 1: Root system of sl₂
> $\mathfrak{sl}_2$ has rank 1 with a single positive root $\alpha$. The root spaces are:
> - $\mathfrak{g}_\alpha = \mathbb{C} e$ (spanned by the raising operator)
> - $\mathfrak{g}_{-\alpha} = \mathbb{C} f$ (spanned by the lowering operator)
>
> The Weyl vector is simply $\rho = \alpha/2$.

> [!hint]- Hint 2: Weyl group
> The Weyl group $W$ of $\mathfrak{sl}_2$ is generated by the reflection $s_\alpha$ corresponding to $\alpha$. Since $s_\alpha^2 = 1$:
> $$W = \{1, s_\alpha\} \cong \mathbb{Z}/2\mathbb{Z}$$
>
> The reflection acts as $s_\alpha(\mu) = \mu - \langle \mu, \alpha^\vee \rangle \alpha$.

> [!hint]- Hint 3: Substitution
> Set $z = e^{\alpha/2}$ so that $e^\alpha = z^2$ and $e^{-\alpha} = z^{-2}$.
>
> Then $e^\rho = e^{\alpha/2} = z$ and $e^{s_\alpha(\rho)} = e^{-\alpha/2} = z^{-1}$.

> [!hint]- Hint 4: Character formula
> For $\mathfrak{sl}_2$, the highest weight of $V_n$ is $\lambda = n \cdot \frac{\alpha}{2}$.
>
> The Weyl character formula gives:
> $$\text{ch } V_n = \frac{e^{(\lambda + \rho)} - e^{s_\alpha(\lambda + \rho)}}{e^\rho - e^{s_\alpha(\rho)}}$$

## Solution

> [!success]- Solution
> **Part (a): Setup for $\mathfrak{sl}_2$**
>
> For $\mathfrak{sl}_2(\mathbb{C})$:
> - **Root system**: $\Delta = \{\alpha, -\alpha\}$ where $\alpha(h) = 2$ for $h = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$
> - **Positive roots**: $\Delta^+ = \{\alpha\}$
> - **Weyl group**: $W = \{1, s_\alpha\}$ where $s_\alpha(\mu) = -\mu$
> - **Weyl vector**: $\rho = \frac{1}{2}\alpha$
>
> ---
>
> **Part (b): Verifying the denominator formula**
>
> The left-hand side is:
> $$\sum_{w \in W} \epsilon(w) e^{w(\rho)} = \epsilon(1) e^{\rho} + \epsilon(s_\alpha) e^{s_\alpha(\rho)}$$
> $$= e^{\alpha/2} - e^{-\alpha/2}$$
>
> The right-hand side is:
> $$e^\rho \prod_{\alpha \in \Delta^+}(1 - e^{-\alpha}) = e^{\alpha/2}(1 - e^{-\alpha})$$
> $$= e^{\alpha/2} - e^{\alpha/2 - \alpha} = e^{\alpha/2} - e^{-\alpha/2}$$
>
> ✓ Both sides equal $e^{\alpha/2} - e^{-\alpha/2}$.
>
> Setting $z = e^{\alpha/2}$, this becomes $z - z^{-1}$.
>
> ---
>
> **Part (c): Character of $V_n$**
>
> The $(n+1)$-dimensional irreducible representation $V_n$ has highest weight $\lambda = \frac{n\alpha}{2}$.
>
> Using the Weyl character formula:
> $$\text{ch } V_n = \frac{\sum_{w \in W} \epsilon(w) e^{w(\lambda + \rho)}}{\sum_{w \in W} \epsilon(w) e^{w(\rho)}}$$
>
> **Numerator**:
> $$\lambda + \rho = \frac{n\alpha}{2} + \frac{\alpha}{2} = \frac{(n+1)\alpha}{2}$$
>
> $$e^{\lambda + \rho} - e^{s_\alpha(\lambda + \rho)} = e^{(n+1)\alpha/2} - e^{-(n+1)\alpha/2}$$
>
> With $z = e^{\alpha/2}$: $z^{n+1} - z^{-(n+1)}$
>
> **Denominator**: $z - z^{-1}$ (from part b)
>
> **Character**:
> $$\text{ch } V_n = \frac{z^{n+1} - z^{-(n+1)}}{z - z^{-1}}$$
>
> Using the geometric series formula (or factoring):
> $$\frac{z^{n+1} - z^{-(n+1)}}{z - z^{-1}} = z^n + z^{n-2} + z^{n-4} + \cdots + z^{-n}$$
>
> This is verified by:
> $$(z - z^{-1})(z^n + z^{n-2} + \cdots + z^{-n})$$
> $$= z^{n+1} + z^{n-1} + \cdots + z^{1-n} - z^{n-1} - z^{n-3} - \cdots - z^{-(n+1)}$$
> $$= z^{n+1} - z^{-(n+1)}$$
>
> The character $z^n + z^{n-2} + \cdots + z^{-n}$ confirms that $V_n$ has:
> - Weights: $n, n-2, n-4, \ldots, -n$ (all with multiplicity 1)
> - Dimension: $n + 1$ terms
