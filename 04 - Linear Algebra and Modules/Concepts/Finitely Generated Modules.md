# Finitely Generated Modules

## Definition

> [!info] Definition (Finitely Generated Module)
> An $R$-[[04 - Linear Algebra and Modules/Concepts/Module Definition|module]] $M$ is **finitely generated** if there exist elements $m_1, \ldots, m_n \in M$ such that:
> $$M = Rm_1 + Rm_2 + \cdots + Rm_n = \langle m_1, \ldots, m_n \rangle$$
> The elements $\{m_1, \ldots, m_n\}$ are called **generators**.

## Key Properties

1. Finite-dimensional [[04 - Linear Algebra and Modules/Concepts/Vector Spaces|vector spaces]] are finitely generated
2. [[04 - Linear Algebra and Modules/Concepts/Quotient Modules|Quotients]] of finitely generated modules are finitely generated
3. Submodules of finitely generated modules may **not** be finitely generated (in general)
4. Over [[04 - Linear Algebra and Modules/Concepts/Noetherian Modules|Noetherian]] rings, submodules of f.g. modules are f.g.

## Nakayama's Lemma

> [!abstract] Nakayama's Lemma
> Let $M$ be a finitely generated $R$-module and $I \subseteq R$ an ideal contained in the Jacobson radical. If $IM = M$, then $M = 0$.
>
> **Corollary**: If $M/IM$ is generated by $\bar{m}_1, \ldots, \bar{m}_n$, then $M$ is generated by $m_1, \ldots, m_n$.

## Structure Theorem

> [!abstract] Fundamental Theorem (over PID)
> If $R$ is a [[02 - Ring Theory/Concepts/Principal Ideal Domains|PID]] and $M$ is a finitely generated $R$-module, then:
> $$M \cong R^r \oplus R/(a_1) \oplus R/(a_2) \oplus \cdots \oplus R/(a_k)$$
> where $a_1 | a_2 | \cdots | a_k$ (**invariant factors**).
>
> Alternatively (elementary divisors):
> $$M \cong R^r \oplus \bigoplus_{i} R/(p_i^{e_i})$$
> where $p_i$ are primes.

## Special Cases

| Ring | Finitely Generated Modules |
|------|---------------------------|
| Field $F$ | Finite-dimensional vector spaces |
| $\mathbb{Z}$ | Finitely generated abelian groups |
| $k[x]$ | Finite-dimensional $k$-vector spaces with linear operator |

## Examples

> [!example] Example 1: Finitely generated abelian groups
> $\mathbb{Z}^n \oplus \mathbb{Z}/d_1\mathbb{Z} \oplus \cdots \oplus \mathbb{Z}/d_k\mathbb{Z}$

> [!example] Example 2: Not finitely generated
> $\mathbb{Q}$ as a $\mathbb{Z}$-module is **not** finitely generated.

> [!example] Example 3: Polynomial ring
> $R[x]$ is **not** finitely generated as an $R$-module (but is as an $R$-algebra).

> [!example] Example 4: Ideal as module
> Every ideal in a [[04 - Linear Algebra and Modules/Concepts/Noetherian Modules|Noetherian ring]] is finitely generated.

## Presentations

> [!info] Definition (Presentation)
> A **presentation** of $M$ is an [[04 - Linear Algebra and Modules/Concepts/Exact Sequences|exact sequence]]:
> $$R^m \xrightarrow{\phi} R^n \xrightarrow{\pi} M \to 0$$
> The matrix of $\phi$ is called a **presentation matrix**.
