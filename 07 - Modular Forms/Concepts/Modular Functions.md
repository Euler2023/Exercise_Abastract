# Modular Functions

## Definition

> [!info] Definition (Modular Function)
> A **modular function** for $\text{SL}_2(\mathbb{Z})$ is a meromorphic function $f: \mathbb{H} \to \mathbb{C} \cup \{\infty\}$ satisfying:
> 1. $f(\gamma \cdot \tau) = f(\tau)$ for all $\gamma \in \text{SL}_2(\mathbb{Z})$
> 2. $f$ is meromorphic at the cusp $i\infty$

Equivalently, $f$ is a meromorphic function on the modular curve $X(1) = \mathbb{H}^*/\text{PSL}_2(\mathbb{Z})$.

## Comparison with Modular Forms

| Property | Modular Function | [[07 - Modular Forms/Concepts/Modular Forms Definition|Modular Form]] |
|----------|------------------|---------------|
| Weight | 0 | $k > 0$ |
| Transformation | $f(\gamma\tau) = f(\tau)$ | $f(\gamma\tau) = (c\tau+d)^k f(\tau)$ |
| At cusps | Meromorphic | Holomorphic |
| On $\mathbb{H}$ | Meromorphic | Holomorphic |

## Field of Modular Functions

> [!abstract] Theorem
> The field of modular functions for $\text{SL}_2(\mathbb{Z})$ is:
> $$\mathbb{C}(X(1)) = \mathbb{C}(j)$$
> a rational function field generated by the [[07 - Modular Forms/Concepts/j-Invariant|j-invariant]].

> [!tip] Consequence
> Every modular function $f$ can be written as $f = P(j)/Q(j)$ for polynomials $P, Q$.

## Construction from Modular Forms

> [!info] Proposition
> If $f \in M_k$ and $g \in M_k$ with $g \not\equiv 0$, then $f/g$ is a modular function.

Examples:
- $j = 1728 E_4^3 / \Delta$ (weight $12/12 = 0$)
- $E_4^3/E_6^2$ is a modular function

## Zeros and Poles

> [!abstract] Theorem (Valence Formula)
> For a non-zero modular function $f$:
> $$\sum_{p \in X(1)} \text{ord}_p(f) = 0$$
> counting multiplicities with weights $1/2$ at $i$, $1/3$ at $\rho$, and $1$ at the cusp.

## Examples

> [!example] Example 1: j-invariant
> The [[07 - Modular Forms/Concepts/j-Invariant|j-invariant]] is the fundamental modular function:
> - Simple pole at $i\infty$
> - Triple zero at $\rho$
> - No other zeros or poles

> [!example] Example 2: $j - 1728$
> - Double zero at $i$
> - Simple pole at $i\infty$
> - Zero of order $-3$ at $\rho$ (since $j(\rho) = 0$)

> [!example] Example 3: Klein's j-function
> $j(\tau) = \frac{1}{q} + 744 + 196884q + \cdots$

## Modular Functions for Subgroups

For [[07 - Modular Forms/Concepts/Congruence Subgroups|congruence subgroups]] $\Gamma$:
- $\mathbb{C}(X(\Gamma))$ is a finite extension of $\mathbb{C}(j)$
- The extension degree equals $[\text{PSL}_2(\mathbb{Z}) : \bar{\Gamma}]$

## Hauptmodul

> [!info] Definition (Hauptmodul)
> A **Hauptmodul** (principal modulus) for a genus 0 group $\Gamma$ is a modular function $t$ generating the function field:
> $$\mathbb{C}(X(\Gamma)) = \mathbb{C}(t)$$

The $j$-invariant is the Hauptmodul for $\text{SL}_2(\mathbb{Z})$.
